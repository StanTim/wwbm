
1.Работа с книгами

  -Пользователь добавляет книгу на сайт.
  -Посетитель ищет книгу с помощью поиска.
  -Пользователь ставит книге оценку.
  -Пользователь отмечает книгу прочитанной.
  -Пользователь отмечает книгу к прочтению.
  -Посетитель видит инфо о книге: количество прочтений книги, средний рейтинг книги, кто добавил, автор и тд.
  -Оставляет комментарий
  -Пользователь может отредактировать/удалить книгу/комментарий/оценку

2.Работа со списками

  -Пользователь создает/удаляет свой персональный список книг
  -Пользователь добавляет книгу в личный список
  -Пользователь видит список пользователей: -- читавших данную книгу. -- которые будут читать книгу.
  -Пользователь видит последние несколько комментариев к книге
  -Пользователь может удалить книгу из списка

3.Работа с пользователями

  -Пользователь ищет других пользователей при помощи поиска.
  -Пользователь открывает профиль другого пользователя где есть списки с инфо о пользователе.
  -Пользователь редактирует свой профиль.
